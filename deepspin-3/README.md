## Training and Evaluation
To train, segment and evaluate use:
```
./train.sh <language code>
```

Language options include:
| Language | Language code |
|----------|---------------|
|English|eng|
|French|fra|
|Hungarian|hun|
|Italian|ita|
|Latin|lat|
|Mongolian|mon|
|Russian|rus|
|Spanish|spa|


output:
```
category: all
distance	0.34
f_measure	95.44
precision	95.05
recall	95.83
```
